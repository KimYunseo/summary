GIT
===

1. VSC(Version Control System)

>파일 변경이력을 기록해 관리를 쉽게 할 수 있도록 해주는 시스템.

2. 이점 

>* 변경이력을 통해 그 내용 공유 가능
>* 타인 작업내용을 쉽게 병합
>* 쉽게 복구 가능
>* Branch를 통해 병렬관리 가능

3. Git 작업 흐름

| working Directory	| Staging Area	| Repository |
|-------------------|---------------|------------|
|           <-   Checkout	 the  project     ---  |
|   	 ---   Stage Fixes       -->	|            |
|                 	|       ---     Commit -->   |
